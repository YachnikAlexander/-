https://belavia.by

1. Форма для заказа билета - 1 этап

	Выбираем раздел Купить
	Заполняем поле ввода Откуда
	Заполняем поле ввода Куда
	Выбираем Тип билета: в одну сторону, туда-обратно
	Дата вылета
	Дата приезда обратно
	Кол-во пассажиров
	Нажать на кнопку Найти - перенаправление на https://ibe.belavia.by/select/MSQABA/2018-12-14/2019-01-14/adults-1/children-0/infants-0 
2. Форма для заказа билета - 2 этап
	Появляется список всех возможных заказов на этот день
	Кликаем на один из них(на цену)
	Выбираем вид билета(ограниченный, полугибкий, гибкий)
	После этого наживаем далее 
3.Форма для заказа билета - 3 этап
	Вводим данные в появившихся формах
		-Обращение
		-имя
		-фамилия
		-гражданство
		-номер паспорта
		-дата рождения
		-дата окончания действия паспорта
	Нажимаем далее
4. Попытка ввести невалидные даные =  имя фамилия
	Вводим в поля ввода Имя, фамилия данные = "888"
	Поля ввода подсвечиваются красным и выводится errorMessage = Может содержать только латинские буквы и пробелы
4. Попытка ввести невалидные даные =  имя фамилия
	Вводим в поля ввода Имя, фамилия данные = "888"
	Поля ввода подсвечиваются красным и выводится errorMessage = Может содержать только латинские буквы и пробелы
5. Попытка ввести невалидные даные в поля =  дата рождения, дата окончания паспорта
	Вводим в поля ввода данные = "рроав"
	Данные не вводятся в поля ввода
6. Оплата заказа
	выбираем способ оплаты
	реконнектит к сервису по оплате
7.	Оплачиваем заказ
8. Ошибка на день вылета(нет нужных билетов)
	Выбираем день вылета
	Выбираем откуда
	Выбираем куда
	Нажимаем "Найти" - идет перенаправление на https://ibe.belavia.by/select/MSQABA/2018-12-14/2019-01-14/adults-1/children-0/infants-0 и сообщение о том, что рейсов в этот день нет
9.Попытка выбрать даты обратно раньше даты туда
	Выбираем день туда(14.12.2018)
	Нажимаем на поле обратно 
	Выбираем 13 декабря - дата не может быть выбрана, так как со стороны клиента блочаться все дни календаря, которые раньше декабря
10.попытка купить билет на 6 взрослых людей
	Выбираем поле ввода "Пассажиры"
	Кликаем на "+" 4 раза
	После 5 взрослых пассажиров не можем дальше выбрать больше, так как со стороны клиента блочится выбор
